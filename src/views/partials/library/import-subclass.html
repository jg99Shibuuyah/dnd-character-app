Import Subclass◈

Add a subclass to an existing parent class (built-in or imported). It becomes selectable on that class in the Classes & Levels picker, and its features show on the Features & Traits and Actions tabs, gated by your class level.
Pick any built-in or imported subclass to load it into the form. Re-importing with the same parent + name overwrites it; a new name saves a copy.
Format: level | name | description | use | cost. Level is the character level the feature unlocks at. use + cost surface it on the Actions tab.
Paste a full subclass definition to import it directly. Overrides the form fields above.
