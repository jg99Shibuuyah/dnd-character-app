Import Class◈

Add a homebrew or official class, using the same fields the built-in Jaeger class uses. Imported classes are saved to the database and shared across every character, then appear in the class picker on the Settings tab.
Pick any built-in or imported class to load it into the form. Re-importing with the same name overwrites it; a new name saves a copy.
Comma-separated skill names the class can choose from. Blank = any skill.
Comma-separated. Chosen at the Subclass Level above.
Format: level | name | description | use | cost | choices. Only level and name are required. use (reaction / bonus action / free / special) and cost surface the feature on the Actions tab. choices is a semicolon-separated list — a feature with choices prompts the character to pick one on the Settings tab. Leave a field empty to skip it (e.g. 2 | Fighting Style | | | | Archery; Defense).
Paste a full class definition to import it directly. Overrides the form fields above.
