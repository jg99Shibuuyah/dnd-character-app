Import Background◈

Add a character background. Imported backgrounds are saved to the database and appear in the Background picker on the Settings tab; their skill proficiencies apply automatically and their feature shows on the Features & Traits tab.
Pick any built-in or imported background to load it into the form. Re-importing with the same name overwrites it; a new name saves a copy.
Comma-separated skill names — these are granted automatically to any character with this background.
