Bulk Import — JSON◈

Paste or load a JSON file holding many entries at once — classes, species, subclasses, subspecies, and spells mixed together. Each entry is saved to the shared database and appears in the matching picker. Entries are deduplicated by type and name: importing a name that already exists (here or in the batch) overwrites it rather than creating a copy.
Accepts an array of entries (each with a type of class / species / subclass / subspecies / spell), a grouped object like { "classes":[…], "spells":[…] }, or a single entry. Subclasses and subspecies need a parent. When type is omitted it is inferred from the fields. Each entry may nest its fields under data or list them inline.
