Import Subspecies◈

Add a subspecies / subrace to an existing parent species (built-in or imported). It becomes selectable under the Subspecies / Subrace picker on that species in Settings, and its traits show on the Features & Traits tab.
Pick any built-in or imported subspecies to load it into the form. Re-importing with the same parent + name overwrites it; a new name saves a copy.
Format: name | description, one trait per line.
Paste a full subspecies definition to import it directly. Overrides the form fields above.
