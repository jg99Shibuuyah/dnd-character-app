How Imports Read JSON◈

Every entry has a name, an optional source, and a data object holding its fields. Subclasses and subspecies also need a parent. The Paste JSON… box on each import form takes a single entry; the Bulk Import tab takes many at once.
  • name required — the display name. Re-importing a name that already exists overwrites it (dedup is by type + name).
  • source — one of "5E", "5E (legacy)", "5.5E", "Homebrew". Anything else defaults to Homebrew.
  • parent subclass / subspecies — the name of the parent class or species it attaches to.
  • data — the object of type-specific fields shown below. Fields may also be listed inline instead of nested under data.
The Bulk Import tab accepts three shapes:
  • An array of entries, each tagged with a type — class, species, subclass, subspecies, or spell.
  • A grouped object keyed by type: { "classes": [ … ], "spells": [ … ] }. No per-entry type needed.
  • A single entry. If type is omitted it is inferred from the fields present.
[
  { "type": "class",  "name": "Artificer", "source": "5E", "data": { … } },
  { "type": "spell",  "name": "Mend Bone", "data": { … } },
  { "type": "subclass", "parent": "Artificer", "name": "Alchemist", "data": { … } }
]

// — or grouped —

{
  "classes":  [ { "name": "Artificer", "data": { … } } ],
  "spells":   [ { "name": "Mend Bone", "data": { … } } ]
}

Character◈

Characters are imported from the sheet page, not this Import page — the Import button in the Profile bar reads a JSON file holding one character, saves it as a new profile, and loads it. The file is either a bare character object (below) or the { "name": …, "data": { … } } wrapper the API returns, so a saved API response imports directly.
{
  "name": "Tessa Brightblade",
  "class": "Fighter",
  "level": 5,
  "race": "Human",
  "background": "Soldier",
  "alignment": "Lawful Good",
  "xp": 6500,
  "classes": [ { "name": "Fighter", "level": 5 } ],
  "abilities": { "str": 16, "dex": 14, "con": 15, "int": 10, "wis": 12, "cha": 8 },
  "saveProf": { "str": true, "con": true },
  "ac": 18, "speed": 30,
  "hpMax": 44, "hpCurrent": 44, "hpTemp": 0,
  "hitDice": "5d10",
  "attacks": [ { "name": "Longsword", "bonus": "+6", "dmg": "1d8+3 slashing" } ],
  "currency": { "cp": 0, "sp": 30, "gp": 25 },
  "inventory": [ { "name": "Rope (50 ft)", "qty": 1 } ],
  "knownSpells": [],
  "persTraits": "I face problems head-on."
}
  • Every field is optional — anything missing gets the new-character default, and imports always create a new profile (an id in the file is ignored).
  • name — the profile name. Missing or blank falls back to Unnamed Adventurer.
  • classes — array of { "name", "level" }; the first entry is the primary class. Keep the flat class / level fields matching the primary entry.
  • abilities — scores keyed str dex con int wis cha.
  • saveProf — saving-throw proficiencies, same six keys, true / false.
  • skillProf — uses internal keys (sk0–sk17, skills in sheet order); easiest to tick on the Skills tab after importing.
  • knownSpells — array of { "name", "level", "tags": [ … ] }.
  • attacks / inventory / currency — as shown; persTraits, persIdeals, persBonds, persFlaws, and features are free text.

Class◇

{
  "type": "class",
  "name": "Artificer",
  "source": "5E",
  "data": {
    "hitDie": 8,
    "saves": ["con", "int"],
    "choose": 2,
    "skills": ["Arcana", "History"],
    "subclassLevel": 3,
    "subclasses": ["Alchemist", "Artillerist"],
    "casting": { "type": "half", "ability": "int" },
    "desc": "A master of invention.",
    "features": [
      { "lv": 1, "name": "Magical Tinkering" },
      { "lv": 2, "name": "Infuse Item", "use": "special", "cost": "2 infusions" },
      { "lv": 3, "name": "Fighting Style", "choices": ["Archery", "Defense", "Dueling"] }
    ]
  }
}
  • hitDie — hit-die size: 6, 8, 10, or 12.
  • saves — array of ability keys: str dex con int wis cha.
  • choose — number of skill proficiencies the class picks.
  • skills — array of skill names to choose from, or "any".
  • subclassLevel — the level a subclass is chosen (default 3).
  • subclasses — array of subclass names (optional).
  • casting — { "type": "none" | "full" | "half" | "pact", "ability": "int" | "wis" | "cha" }.
  • features — array of { lv, name, desc?, use?, cost?, choices? }. use (reaction / bonus action / free / special) and cost surface it on the Actions tab.
  • choices — array of option strings on a feature. A feature with choices prompts the character to pick one on the Features and Settings tabs, with an Other… option that opens a popup for naming a custom one. Built-in Fighting Style features are filled in automatically with the styles that class can take.
  • desc — one-paragraph summary (optional).

Subclass◇

{
  "type": "subclass",
  "parent": "Artificer",
  "name": "Alchemist",
  "source": "5E",
  "data": {
    "subclassLevel": 3,
    "desc": "Potions and elixirs.",
    "features": [
      { "lv": 3, "name": "Experimental Elixir" }
    ]
  }
}
  • parent required — the class this attaches to.
  • subclassLevel — defaults to the parent class's subclass level.
  • features — array of { lv, name, desc?, use?, cost?, choices? }, same as a class.
  • desc — one-paragraph summary (optional).

Species◇

{
  "type": "species",
  "name": "Aarakocra",
  "source": "5E",
  "data": {
    "size": "Medium",
    "speed": 30,
    "darkvision": 0,
    "asi": "+2 Dex, +1 Wis",
    "languages": "Common, Auran",
    "desc": "Bird-folk of the peaks.",
    "traits": [
      { "name": "Flight", "desc": "Flying speed 50 ft." }
    ]
  }
}
  • size — e.g. "Small", "Medium".
  • speed — walking speed in feet (default 30).
  • darkvision — range in feet, or 0 for none.
  • asi — ability score increases, free text (optional).
  • languages — free text (optional).
  • traits — array of { name, desc? }.
  • desc — one-paragraph summary (optional).

Subspecies◇

{
  "type": "subspecies",
  "parent": "Elf",
  "name": "High Elf",
  "source": "5E",
  "data": {
    "asi": "+1 Int",
    "desc": "Attuned to arcane magic.",
    "traits": [
      { "name": "Cantrip", "desc": "Know one wizard cantrip." }
    ]
  }
}
  • parent required — the species this attaches to.
  • traits — array of { name, desc? }.
  • asi — ability score increases, free text (optional).
  • desc — one-paragraph summary (optional).

Spell◇

{
  "type": "spell",
  "name": "Fireball",
  "source": "5E",
  "data": {
    "level": 3,
    "classes": ["Sorcerer", "Wizard"],
    "school": "Evocation",
    "castingTime": "1 Action",
    "range": "150 feet",
    "components": "V, S, M",
    "duration": "Instantaneous",
    "tags": ["Damage", "AoE"],
    "desc": "A bright streak of flame."
  }
}
  • level — 0–9 (0 = cantrip).
  • classes — array of class names that can cast it. Empty = every class.
  • school, castingTime, range, components, duration — free text (optional).
  • tags — array of labels (e.g. "Reaction" surfaces it on the Actions tab).
  • desc — the spell's effect text (optional).
